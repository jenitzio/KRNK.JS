# **KRNK.JS**
A fast-paced, competitive, browser-based FPS inspired by Krunker.io. This game is built entirely using HTML5, CSS3, and JavaScript. It requires no downloads or plugins and is designed to be played from anywhere via a central server connection.
Features

### Instant Play: 
No installation required. Playable directly in the browser.

### Real-time Multiplayer: 
Low-latency gameplay powered by Node.js and Socket.io.

### 3D Rendering: 
Built using Three.js and WebGL for high-performance graphics.

### Movement Mechanics: 
Features slide hopping, flick shots, and physics-based movement.

### Global Connectivity: 
Lightweight server architecture allows users to join from anywhere.


## Tech Stack

### Frontend: 
HTML5, CSS3, JavaScript

### Rendering Engine: 
Three.js

### Backend: 
Node.js

### Networking: 
Socket.io / WebSockets for bi-directional communication.

### Physics: 
Cannon.js


## How to Play

### Access the Game: https://krnk-io.wasmer.app/

### Controls:
WASD: Movement
Space: Jump
Shift: Crouch/Slide
Left Click: Shoot
Right Click: Aim Down Sights (ADS)
R: Reload


## How to Run the Game
To run the game, you do not need to follow a complex installation process. Use one of the following methods:
Online HTML Compiler: Copy the source code and paste it into any online HTML/JavaScript compiler (such as CodePen, JSFiddle, or Programiz) to run it instantly in your browser.
Local Execution: Download the source files to your computer and open the index.html file in any modern web browser to start the game locally.
